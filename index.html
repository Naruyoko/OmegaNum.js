OmegaNum.js

A large number library reaching 10{1000}10 in Bower's operator notation.
For the code, you can find it here, and you can clone it or copy and paste it or whatever to use it.

How it works

This library works by storing the sign and an array.
The sign is either 1 or -1, positive or negative respectively.
Array holds the magnitude; Array [n0,n1,n2,n3...] represents:
...(10↑3)n3(10↑↑)n2(10↑)n1n0
Therefore, [23.2352,2,6,0,1,4] represents (10↑5)410↑4(10↑↑)6(10↑)223.2352, which without parenthesis becomes 10↑510↑510↑510↑510↑410↑↑10↑↑10↑↑10↑↑10↑↑10↑↑10↑10↑23.2352. Using LaTeX, I can sort of expands into decimal, which is:
