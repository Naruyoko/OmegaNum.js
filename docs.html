absoluteValue abs

Returns the absolute value of method. 


        console.log(OmegaNum("10^^ee5").abs().toString()) //"10^^ee5"
        console.log(OmegaNum.abs(-3.7e52).toString()) //"3.7e52"

arrow

For static method, it is in form OmegaNum.arrow(x,arrows,y), while non-static method is x.arrow(arrows)(y). 


        console.log(OmegaNum(13).arrow(3)(5).toString()) //"(10^^)^3 (10^)^11 337385711567664.9"
        console.log(OmegaNum.arrow(6,4,5).toString()) //"(10^^^)^3 (10^^)^4 eeee36305.31580191892"
For non-static method, not having the second parameter will return function of the arrows.

ceiling ceil

Returns the ceiling of method. 


        console.log(OmegaNum(2.34).ceil().toString()) //"2.34"
        console.log(OmegaNum.ceil(-24.32).toString()) //"-24.32"

compareTo cmp

Compares two numbers, returns 1 if greater, 0 if equal, and -1 if less than. 


        console.log(OmegaNum(5).cmp(3)) //1
        console.log(OmegaNum.cmp(-8,3)) //-1
        console.log(OmegaNum.cmp(6,6)) //0

cubeRoot cbrt

Returns the cube root of the number. 


        console.log(OmegaNum(5).cbrt().toNumber()) //29.172491415642423
        console.log(OmegaNum.cbrt(-4).toNumber()) //Infinity
        console.log(OmegaNum.cbrt(3).toNumber()) //10

divide div

        OmegaNum.div(6,2).toString() //"3"
        OmegaNum.div("ee5",2).toString() //"4.999999999978076e99999"
        OmegaNum.div("ee5","-10^^3").toString() //"-0"

equals eq

Returns whether two numberws are equal. 


        OmegaNum.div(5,5) //true
        OmegaNum.div(0,120) //false
        OmegaNum.div("10^^e16",[16,1,1])] //true
